# Sync your Storybook components with Figma

Generate a Figma library file out of your storybook components to create a single source of truth for designers and developers.

Import this sample storybook to Figma, by clicking the “Start sync” button on the [**Anima Figma plugin**](https://www.figma.com/community/plugin/857346721138427857) under "Manage Design Systems" option.
